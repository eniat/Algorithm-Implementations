Q. (i.) Design an in place algorithm for removing all repeated elements from an array containing n integers in the range from 0 to 1000 that runs in Theta(n) time in the worst case and *does not use hashing*. There may be satellite data associated with the integer keys and your algorithm should only remove repeated elements that occur later in the sequence, retaining those that appear first. Write your algorithm in pseudocode.

Answer:
In Python code as stated in can be in the question:

def duplicate_remove(A):

    # Initializing B array with -1's to track first occurrences 0-1000
    B = [-1] * (1000 +1)

    # Keeps the position to write unique elements to
    i = 0

    # Goes through the A array checking if a number is unique and moves it to the front
    for num in A:
        if B[num] == -1: #Checking it's the first occurrence
            B[num] = i # Store position
            A[i] = num # move it to the front/ new position
            i +=1 # Move on

    # removes all the excess duplicates
    del A[i:]

    return A

Q. (v.) (a) Give precise definitions of a full binary tree and a complete binary tree. How are these notions different?

Answer:
A full binary tree is a binary tree where each node is either a leaf or has degree exactly 2. This means that every parent node or internal node has either two or no children making them either a leaf node or have exactly 2 children. A complete binary tree is a tree where all of the levels of the tree are completely filled except for the lowest level and on the lowest level they are filled from as left as possible to right. This means that the differences are a complete binary tree can have a node with only 1 child where a full binary tree can not. A complete binary tree is often used in heap based data structures where as a full binary tree has no similar application. A full binary tree can be filled in any order where as a complete binary tree has to be filled from left to right.

Q. (vi.) (a)  Give a definition of the heap data structure and state its main properties.

Answer: 
A heap is a complete binary tree structure that also satisfies the heap property. There are two main heap properties the first being max heap which means the value of each parent node is greater than or equal to the values of it's children and min heap where the value of each parent node is less than or equal to the values of its children. The heap also stores a collection of objects with its keys. A heap is normally implemented as an array where each node in the tree corresponds to an element of the array. arr[(i-1)/2] returns the parent node, arr[(2*i)+1] returns the left child node and finally arr[(2*i)+2] returns the right child node.
